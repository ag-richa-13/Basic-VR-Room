## 5. Creating the VR Room Scene

1. **File â–¸ New Scene** â†’ *3D (URP/HDRP/BiRP)* â†’ save as `Room.unity` inside `Assets/Scenes`.
2. **GameObjects** to create:
   * *Room_Root* _(empty parent)_ positioned at *(0,0,0)* â€“ this will be the anchor for all room content.
   * *XR Origin (Action-Based)* â€“ via **GameObject â–¸ XR â–¸ XR Origin (Action-based)**.
   * *Directional Light* â€“ for basic lighting.
3. Drag the **FurnitureSpawner** script onto *Room_Root* and, in the Inspector:
   * Assign **Chair Prefab** (`Assets/Prefab/Chair.prefab`).
   * Assign **Table Prefab** (`Assets/Prefab/Table.prefab`).
   * Set **Room Root** to the *Room_Root* transform.
4. Tweak **chairPosition / tablePosition** & **Rotation** fields if you want to move furniture around.

> Tip: Hold **Ctrl + Shift** and drag your headset/controller in the Scene view to test the furniture placement quickly.

## 6. Running in VR

1. Connect your headset (ensure SteamVR / Oculus software is running).
2. Press **Play** in the Unity Editor; the *XR Origin* will take control and you can move around using thumb-sticks.
3. The **FurnitureSpawner** script will instantiate the **Chair** & **Table** inside the room automatically at runtime.

## 7. Building the Project

1. **File â–¸ Build Settings**.

## 8. Customising Furniture

1. **Create 3-D Models / Prefabs** for additional furniture.
2. Drop them into `Assets/Prefab/`.
3. Update `FurnitureSpawner` with new public `GameObject` references and placement transforms.

```csharp
// Example addition inside FurnitureSpawner

public Vector3 lampPosition = new Vector3(0,0,2);
public Vector3 lampRotation = Vector3.zero;
```
